ARITHMETIC OPERATIONS

Arithmetic Operators

To make calculations with values and variables, we use arithmetic operators. Each data type has its own set of operators, but since many of the primitive types overlap as integer values, the operators also mostly overlap. In this lesson we will give an overview of the main operators used in C++ ordered by data type.

Integers

The operators [+, -, * (multiplication)] work as you would expect them to, and it’s used like the following:


int x = 5 + 3;
// or
int x = y + z;
// or
cout << x+y << "\n";

where y and z are integers, and the + can be replaced with any of the other operators.

Division is done using the ‘/’ symbol, but it's a little different than regular mathematical division. When C++ divides two integers it wants the answer to also be an integer so it truncates everything past the decimal point, so 5 / 3 = 1 not 1.66.

There is also another standard operator ‘%’, the remainder operator. The value of a % b is the remainder when a is divided b. For instance, 13 % 5 = 3.

The one restriction for integer operators is that we cannot divide by 0.

Characters

Recall that characters are also integers; operators on characters are the same as on integers. However, expressions will not be evaluated back to characters; they will become integers. We can turn it back into characters through type casting, but we will deal with this later.

Doubles

The same operators that work on integers work on doubles, except for % (remainder). Divisions of real numbers don’t have remainders so the modulo would always be 0, which is useless. Remember that all operations on two doubles will result in a double as well.

Booleans

Any expression that evaluates to either true or false is a boolean expression. The common boolean operators are && (and), || (or), ! (not). These are exactly the same as the logical expressions you have learned in geometry class, so true and false = false.

Strings

The only operator that works on strings is addition (+), which glues (concatenates) the two strings together. Note that it only works after we store the strings inside of variables. In other words, trying to evaluate “hi” + “bye” doesn’t work. We need to do the following:


string hi = “hi”;
string bye = “bye”;
string hibye = hi + bye;

To get the concatenated string.

Operators On Different Data Types

In the meeting we showed experimentally that C++ has the following order of preference char < int < double. This means that whenever two or three of these types are used in the same arithmetic expression, the final result will be represented as the datatype with the highest preference. So 5 / 3.0 = 1.66 not 1 and 10000 / ‘a’ = 103 not ‘g’.

Strings also take preference over chars when they are added together, i.e. you can add a char to a string by doing the following:


string s = “abc”;
char c = ‘d’
string s2 = s+c;

But also notice that you can’t add a string to a char because a char is limited to one value.

There are four more boolean operators: > (greater than), >= (greater than or equal to), < (less than), <= (less than or equal to). These can be used to compare two compatible data types and return a true or false value. For example (1 <= 3) evaluates to true.

If Statements

The basic syntax of an if statement is the following:


if (condition) {
    // do something here
}

Where condition is a boolean expression, i.e. a >= b or 5 <= 3 or true && false. If this condition expression evaluates to true then the code inside the curly braces is run otherwise it is completely skipped.

There is another form of the if statement that looks like the following:


if (condition) {
    // do something here
} else {
    // do something else here
}

The way this form works is that if the condition is true it runs the code inside the first pair of curly braces, but if it's false it runs the code inside the second pair of curly braces.

Problem: Write a program that takes the age of a user (an integer) as an input, and then outputs “Yes” if they can drink or “No” if they can’t drink. Remember the legal drinking age is 21.

Solution 

#include <bits/stdc++.h>
using namespace std;

int main() {
    int age;
    cin >> age;
    if (age >= 21) {
        cout << "Yes";
    } else {
        cout << "No";
    }
}

Here is why this works: the code inside of the if statement is the "yes" branch. If it's true that the age is greater than or equal to 21, then it executes, printing "Yes". Otherwise, it goes to the code inside of the else block, which is the "no" branch. If the age is not greater than or equal to 21, it would print "No".
