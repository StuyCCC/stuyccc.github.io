Yer A Wizard, ‘Array!

Array Basics

We’ve talked about “sequences of numbers” a lot, and that’s what arrays represent—an ordered sequence of numbers, or strings, or really anything. In theory, they look something like this: A = [a1, a2, … , an], where n is the length of the array.

Arrays can only hold objects of a single data type. We must specify what that type is, its size, and its name, as such: type name[size]; For instance, an integer array of size 50 named “arr” is defined like this: 


int arr[50];

To access the element at the k-th position of the sequence, we can use arr[k]. This can be either changed or used any way you like, and can be treated like a variable. For instance, if you wanted to add 2 to arr[k] and print the sum of arr[k] and arr[k-1], we can do


arr[k] += 2;
cout << arr[k] + arr[k-1] << ‘\n’;

Zero-Indexing

One key thing to remember about arrays is that they are 0-indexed and not 1-indexed. When we order a list of numbers we usually say this is the first number, then the 2nd number, then the third, and so on. That is called 1-indexing because we start our counting at 1. So now what do you think 0-indexing is?

0-indexing means that arrays start at 0 and not 1 so this means our valid index positions are 0 to 49 if we create an array of size 50. If we go below 0 or above 49 then we will get something that we don’t want.

Iteration

The main way to go through an array is using a for loop like the following:


for (int i = 0; i < n; i++) {
    // do something to arr[i].
}

Where n is the size of the array. Remember if you are given an array of size n, the valid indexes are from 0 to n-1, so this for loop is going through all those valid positions. But there is another way:


for (int i : arr) {
    // do something to i.
}

In this example we go through all the ints in the array and our variable i gets represented as those ints.

However, the key difference between this and the first one is that if we do i = 5, this won’t affect the array, but if we do arr[i] = 5 in the first example this will change the array.

N Dimensional Arrays

What we just discussed are 1-dimensional arrays, or simple lists. What if we wanted a 2-dimensional grid, or 3-dimensional box of numbers?

We can initialize a 2-d array of integers like so: 


int arr[10][10];

Which creates a 10 by 10 box. We can create arrays with more dimensions just by putting more brackets with numbers at the end.

We can think of a 2-d array as a 1-d array of 1-d arrays. We can think of the big array of arrays as going from up to down, and each constituent array as going from left to right. Here is an image for convenience:

The syntax for accessing elements is similar: we use arr[i][j]. The first index, i, refers to the index in the bigger array of arrays. The second index, j, refers to the index in the constituent array, arr[i]. You can also use other coordinate systems to think about arrays in 2 dimensions or higher dimensions, as long as it stays consistent.

Extra Tips And Tricks

You can get the size of an array by doing the following:


int arr[10];
int n = sizeof(arr)/sizeof(arr[0]);
// n will be 10.

What this code is doing is that it finds the total amount of memory for the entire array and then the total amount of memory for a single element and then divides the two to get the number of elements. In most circumstances you don't have to worry about not knowing the size of an array, as they are usually provided, but this is just in case you do.

Notice the last element of an array is in the position length - 1 of the array, this is helpful in many cases.

You can initialize arrays by creating a comma separated list of elements surrounded by curly braces like the following: 


int arr[5] = {1, 2, 3, 4, 5};

Or you can loop through the array and set the value of each element in the loop from standard input like the following: 


int n; cin >> n;
int arr[n];
for (int i = 0; i < n; i++) cin >> arr[i];

Warning! Arrays are very easy to mess up, you can do a whole number of things like going out of the valid ranges of an array and messing stuff up, so just be careful that your accessing the correct elements of an array in your code.

Warning! Don't pass arrays to functions unless you really have to. They are a pain to deal with in functions.

Problem 1: Given an array of ints with a known size from standard input, print true if the array is length 1 or more, and the first element and the last element are equal, else print false.

Solution 

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n; cin >> n;
    int arr[n];
    for (int i = 0; i < n; i++) cin >> arr[i];
    if (n == 0) cout << "false\n";
    else if (arr[0] == arr[n-1]) cout << "true\n";
    else cout << "false\n";
}

Problem 2: Given an array of known size from standard input, print the difference between the largest and smallest values in the array. Note: the built-in min(v1, v2) and max(v1, v2) methods return the smaller or larger of two values.

Solution 

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n; cin >> n;
    int arr[n];
    for (int i = 0; i < n; i++) cin >> arr[i];
    int mn = 1000000000, mx = -1000000000;
    for (int i = 0; i < n; i++) {
        mn = min(mn, arr[i]);
        mx = max(mx, arr[i]);
    }
    cout << mx-mn << "\n";
    return 0;
}

Problem 3: Given an array of known size from standard input, print the sum of the numbers in the array, except ignore sections of numbers starting with a 6 and extending to the next 7 (every 6 will be followed by at least one 7). Print 0 for no numbers.

Solution 

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n; cin >> n;
    int arr[n];
    for (int i = 0; i < n; i++) cin >> arr[i];
    int sm = 0;
    bool add = true;
    for (int i = 0; i < n; i++) {
        if (arr[i] == 6) add = false;
        if (add) sm += arr[i];
        if (arr[i] == 7) add = true;
    }
    cout << sm << "\n";
    return 0;
}

Problem 4: Given an array of ints of known size from standard input, print true if the number of 1's is greater than the number of 4's, else print false.

Solution 

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n; cin >> n;
    int arr[n];
    for (int i = 0; i < n; i++) cin >> arr[i];
    int c1 = 0, c4 = 0;
    for (int i = 0; i < n; i++) {
        if (arr[i] == 1) c1++;
        if (arr[i] == 4) c4++;
    }
    if (c1 > c4) cout << "true\n";
    else cout << "false\n";
    return 0;
}

Problem 6: Given an n by m 2D array, arr, create a new array of size n, sum, where sum[i] contains the sum of elements of arr[i].

Solution 

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, m; cin >> n >> m;
    int arr[n][m];
    for (int i = 0; i < n; i++) {
        for (int q = 0; q < m; q++) {
            cin >> arr[i][q];
        }
    }
    int sum[n];
    for (int i = 0; i < n; i++) {
        int sm = 0;
        for (int q = 0; q < m; q++) {
            sm += arr[i][q];
        }
        sum[i] = sm;
    }
    for (int i = 0; i < n; i++) {
        cout << sum[i] << " ";
    }
    cout << "\n";
}
