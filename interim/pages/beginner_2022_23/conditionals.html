If statement

  • Make a decision on whether or not to do something based a condition. Eg. Is a person allowed to drink? If age >= 21, yes, otherwise no.
  • Notice the parts of that statement above. The condition is "age >= 21". The consequence is "yes" or "no", depending on if the condition is satisfied.
  • We can easily translate the statement above into C++: 
                                
    if (age > 21) {
        printf("Yes, they are allowed to drink.");
    }
    else if (age == 21) {
        printf("They are recently allowed to drink.");
    }
    else {
        printf("No, they are not allowed to drink.");
    }
  • We call the stuff inside the parenthese "boolean expressions".

Boolean expressions and operators

  • AND: &&
  • OR: ||
  • EQUALS: ==
  • NOT: !
  • NOT EQUALS: !=
  • >, >=, <, <= works the same way as in math.

Boolean operators precedence (order of operations): NOT > AND > OR ! > && > ||

Some examples:

  • if (weather == "Sunny")
  • if ( (day == "Monday") || (day == "Tuesday" && hour == 8) )
