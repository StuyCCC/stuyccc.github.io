Variable Types

  • Integer int i = 3;
  • Double double d = 3.1415;
  • Character char ch = 'a';
  • String string str = "abcdef";
  • Boolean bool b = true; bool b = false;

Converting between types

Variables can be converted from one type to another fairly easily. The first way is using the equal sign (assignment operator). 

                
double x = 9.9;
int y;
y = x;
// x = 9.9, y = 9 (after converting from integer x)

Casting

Alternatively, you can manually "cast" a variable to the desired data type.

                
double x = 9.9;
int y;
y = (int)x;
// x = 9.9, y = 9 (after being cast from integer x)
