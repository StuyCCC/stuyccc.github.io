EXPLANATION

The problems found in the links below are posted (almost) weekly, but if you want them daily you can check out our discord. Explanations are always posted right below the problem link, but solution code is not given, you have to program this yourself from the explanation.

Problems in the easy difficulty are suitable for the beginner section and the intermediate section.

Problems in the hard difficulty are suitable for the intermediate section and the advanced section.

Remember the most important thing is having fun while solving these problems, don't stress yourselves out, if you can't solve them.
